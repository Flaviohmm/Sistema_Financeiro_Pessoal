# 💰 Finance System

Sistema financeiro pessoal fullstack

## 🚀 Tecnologias
- Java + Spring Boot
- React + Vite
- PostgreSQL

## 📦 Funcionalidades (MVP)
- Dashboard com resumo financeiro
- Estrutura backend com entidades principais

## ▶️ Como rodar

### Backend
./mvnw spring-boot:run

### Frontend
npm install
npm run dev
